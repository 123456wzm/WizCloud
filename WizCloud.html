💡 CORS 跨域说明
浏览器直接请求 WebDAV 可能遇到跨域限制，请任选一种方式解决：
1. 开启上方「CORS 代理」，启动本地代理服务器
   node proxy.js（Node.js）或 python proxy.py（Python）
2. 在 WebDAV 服务器端配置 CORS 头
3. 安装浏览器 CORS 扩展（如 CORS Unblock）
4. Chrome 启动时加 --disable-web-security 参数
